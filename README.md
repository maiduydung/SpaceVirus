# SpaceVirus
Made with love and pygame

## Install
Pygame is required for both Linux and Windows<br/>

### Linux
 ```
 sudo apt-get install python3-pip python3-dev  
 sudo pip3 install pygame
 ```
 ### Windows 
```
pip install pygame
```
<br/>

## Running the game
```
python3 SpaceVirus.py
```
